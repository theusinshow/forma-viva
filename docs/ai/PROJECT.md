## Client

Atelier Forma Viva — escritório/atelier fictício de arquitetura contemporânea, base na região de Santa Catarina. Foco em arquitetura residencial, interiores de alto padrão, reformas autorais e conceitos com forte relação entre luz, matéria e proporção. (Projeto demo para portfólio Coded by M.)

## Business

Captação de projetos autorais de alto padrão. O site é vitrine de autoridade e qualidade — não venda transacional. Diferencial: repertório visual, processo claro e estética editorial superior à média de sites de arquitetos.

## Primary Conversion

Contato qualificado: formulário breve (`/contato`) + canais diretos (e-mail, WhatsApp, Instagram). Tom não-agressivo, alinhado a cliente de alto padrão.

## Primary User Journey

### User
Cliente potencial de alto padrão (casal/família construindo ou reformando) ou profissional buscando referência. Sensível a estética, processo e confiança.

### Need
Validar se o atelier tem repertório, sensibilidade e método para conduzir um projeto autoral — antes de iniciar conversa.

### Entry source
Indicação, Instagram, busca local ("arquiteto residencial alto padrão SC"), referência de portfólio.

### Expected path
Home (hero galeria) → Projetos em destaque → Case individual → Processo / Atelier → Contato.

### Main objections
"Será que entendem meu estilo?", "Têm processo ou só estética?", "É de alto padrão de verdade?", "Vão me ouvir?". Respondidas por: cases fortes, página Processo, voz da página Atelier.

### Conversion action
Enviar formulário de contato ou acionar WhatsApp/e-mail.

### Abandonment risks
Lentidão (imagens pesadas), navegação confusa, tom comercial agressivo, UI que compete com as fotos, conteúdo genérico. Mitigados por performance, clareza e disciplina visual.
